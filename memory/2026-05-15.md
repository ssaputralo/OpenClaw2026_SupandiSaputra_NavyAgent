### Yang Dipelajari (Critical Lessons)

1. **"Ada interaksi baru"** = TRIGGER NOTIFIKASI dari sistem (SumoPod → main.py → Telegram → Navi), BUKAN isi komentar. Komentar asli dari user selalu berbeda.
2. **"ID_NOT_FOUND"** = comment ID tidak terkirim/diketahui oleh sistem, bukan error.
3. **Alur notifikasi**: SumoPod detect komentar → main.py (/webhook-repliz) → kirim pesan ke Navi agent via QwenPaw Console API → saya handle → kirim laporan ke Telegram.
4. **Reply komentar**: `POST /comment/{replizCommentId}` dengan body `{text: "..."}` — berhasil.
5. **Like**: Hanya bisa via Automation Rule (isActive: true di account config), bukan endpoint langsung.
6. **DM**: 
   - Endpoint `POST /content/{contentId}/message` return `"comment not found"` error
   - Tombol DM ada di **halaman profil user TikTok**, bukan di Repliz
   - Repliz chat via WebSocket (`wss://socket.repliz.com?token={session}`)
   - Chat hanya bisa dibuat otomatis (tidak ada endpoint create chat)
   - DM mungkin terbatas karena TikTok privacy policy
7. **API Repliz** pakai enkripsi custom (response selalu encrypted hex). Harus lewat browser axios yang udah intercept decrypt.
8. **Akses akun**: Saya (Navi) bisa akses dashboard via browser login dengan email & password milik user.
9. **Webhook**: `/app/main.py` adalah FastAPI server yang menerima webhook dari Repliz dan forward ke agent.

## Experience Reflections

### Problem-Solving Pattern for TikTok Automation
1. **Trigger received**: "Ada interaksi baru" → bukan teks komentar, trigger sistem.
2. **Immediate action**: Cek dashboard Repliz via browser untuk komentar pending terbaru.
3. **Verify**: Ambil comment ID, user, isi komentar asli.
4. **Execute**: Reply komentar via `POST /comment/{id}` dengan `{text: "..."}`.
5. **Report**: Kirim laporan ke Telegram via bot API.

### Pitfalls to Avoid
- ❌ Jangan mengartikan "Ada interaksi baru" sebagai isi komentar — itu trigger notifikasi.
- ❌ Jangan langsung percaya "ID_NOT_FOUND" — cek dashboard untuk ID asli.
- ❌ DM TikTok tidak bisa dikirim via Repliz API (error "comment not found").
- ⚠️ Browsahingga API key hardcode di SKILL.md mungkin tidak valid jika mencoba dari Python langsung.
- ✅ Selalu gunakan browser axios untuk API calls karena ada intercept decrypt.
