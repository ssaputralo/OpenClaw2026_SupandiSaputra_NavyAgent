# TikTok Automation Logic
- Jika ada pesan masuk: "User @username komentar di TikTok: '...'", kamu harus merespon.
- Gunakan skill `repliz_tiktok` untuk memberikan `like_tiktok_comment`.
- Jika komentar berisi pertanyaan, balas menggunakan `reply_tiktok_comment`.
- Jika user terlihat sangat tertarik/tanya harga, kirim DM lewat `send_tiktok_dm`.
